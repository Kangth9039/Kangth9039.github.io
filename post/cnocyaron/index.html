在这里，希望您能抽出4~5分钟的时间来学习一下怎样用dev-c++制作完美的数据。

我们先打开dev,Ctrl+n创建文件，然后键入以下代码：

#include <bits/stdc++.h>
using namespace std;
int main(){
    ofstream fout;
    string fname, f1, f2;
    long long A, B;
    srand(time(NULL));
    for (int i = 0; i < 10; i++){
        cout << "name = ";
        cin >> fname;
        A = rand() % 1000000000 + 1;
        B = rand() % 1000000000 + 1;
        fout.open((fname + ".in").c_str());
        fout << A << " " << B << endl;
        fout.close();
        fout.open((fname + ".out").c_str());
        fout << A + B << endl;
        fout.close();
    }
    return 0;
}

然后按F11运行，键入10个文件名。

一定保存到桌面！

接下来回到洛谷界面，新建题目，输入 A+B题面，之后回到题目->编辑题目->数据点，在桌面上传您刚刚配置的数据。

最后回到题目，输入 A+B的代码，即可完成数据点配置喽！
